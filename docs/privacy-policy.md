# プライバシーポリシー - 帰ってきたDarkBlueテーマ(X)

最終更新日: 2026年2月18日

## 概要

「帰ってきたDarkBlueテーマ(X)」（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報を一切収集しません。

## 使用する権限

### storage
- トグル（有効/無効）の状態を保存するために使用します
- 保存されるデータはブール値（true/false）1つのみです
- Chrome の同期ストレージを使用し、同一アカウントのデバイス間で設定が同期されます

### activeTab
- ポップアップからアクティブなタブの状態を確認するために使用します
- 現在のタブが X のページかどうかを判定し、テーマの適用状態を表示します

### ホスト権限 (x.com, twitter.com)
- X のページにテーマ変換用の CSS を適用するために必要です
- これらのサイト以外では一切動作しません

## データの収集

本拡張機能は以下のデータを**一切収集しません**:
- 個人情報（氏名、メールアドレス等）
- 閲覧履歴
- Cookie
- 位置情報
- アクセス解析データ

## 外部通信

本拡張機能は外部サーバーとの通信を**一切行いません**。すべての処理はブラウザ内で完結します。

## データの保存

Chrome の `chrome.storage.sync` API を通じて、トグルの有効/無効状態（ブール値1つ）のみを保存します。このデータは Chrome のアカウント同期機能を通じて同期されますが、外部には送信されません。

## 第三者への提供

本拡張機能はデータを収集しないため、第三者への提供も行いません。
